# Портфолио учебных проектов программы Data Science+ на платформе Яндекс.Практикум 

Здесь представлены некоторые проекты, выполненные мною во время учебы в Яндекс Практикуме по специальности "Специалист Data Science расширенный".

Работ было выполнено много - тут я представила лучшие из освещенных аспектов и потребностей МО

| № | Название проекта | Описание | Инструменты| 
|:-------------------|:---------------------- | :---------------------- | :---------------------- |
|1| [Система прогнозирования HR-рисков: предсказание удовлетворенности и оттока персонала](HR_outflow) | Разработка комплексной системы машинного обучения для HR-департамента, направленной на решение двух ключевых задач кадрового менеджмента |*matplotlib, numpy, pandas, seaborn, shap, lightgbm, phik, sklearn* |
|2| [Предсказание стоимости жилья в Калифорнии](Home_Price_Prediction) | Проект решает классическую задачу регрессии — предсказание стоимости жилья в Калифорнии на основе данных 1990 года с использованием Apache Spark и MLlib для эффективной обработки и моделирования |*Apache Spark 3.x, PySpark, MLlib, Spark SQL* |
|3| [Система оценки риска ДТП для каршеринга](Assessment_risk_DTP) | Разработка системы оценки вероятности ДТП по выбранному маршруту движения для каршеринговой компании. Система должна предупреждать водителей о высоком уровне риска и предоставлять рекомендации по маршруту |*SQL, sqlalchemy, sklearn, catboost, lightgbm, PyTorch* |
|4| [Прогноз заказов такси в час](Taxi_order_forecast) | Система предсказания количества заказов для такси в ближайший час для своевременного привлечения водителей | *pandas, numpy, catboost, scikit-learn, statsmodels*|
|5| [Очистка от токсичных комментаривев](Toxsik_comments) | Разработка модели для обнаружения токсичности в комментариях|*pandas, numpy, re, nltk, wordcloud, sklearn*|
